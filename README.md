# Sistema de Recomendación de Películas

## Descripción del Proyecto

Este proyecto consiste en el diseño de un sistema de recomendación de películas basado en el uso de algoritmos híbridos y bases de datos en grafos. El objetivo principal es ayudar a los usuarios a encontrar contenido relevante de manera rápida y personalizada, reduciendo el tiempo de decisión al elegir qué película ver.

El sistema combina dos enfoques principales:

- Filtrado colaborativo, que identifica usuarios con gustos similares para sugerir contenido que ha sido bien valorado por ellos.  
- Filtrado por contenido, que analiza características de las películas como género, actores y director para encontrar similitudes.

Ambos enfoques se integran mediante una base de datos en grafos, donde los usuarios, películas y sus atributos son representados como nodos, y sus relaciones permiten recorrer conexiones para generar recomendaciones más precisas.

---

## Objetivo

Desarrollar el diseño de un sistema capaz de generar recomendaciones personalizadas utilizando:

- Información del comportamiento del usuario  
- Características del contenido  
- Relaciones entre entidades modeladas en un grafo  

---

## Enfoque del Sistema

El sistema implementa un modelo híbrido, que:

1. Analiza las películas vistas y calificadas por el usuario  
2. Identifica usuarios con preferencias similares  
3. Explora relaciones en el grafo (géneros, actores, directores)  
4. Genera recomendaciones basadas en múltiples criterios  
5. Ordena los resultados según su relevancia
